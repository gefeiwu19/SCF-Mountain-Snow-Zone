# Figures

This folder contains manuscript figures generated by the analysis scripts in `code/`.

Figures are not tracked by Git (see `.gitignore`) because they can be regenerated by running the corresponding scripts. To reproduce any figure, run the script listed in the table below.

## Figure Inventory

| Figure | Filename | Generated by | Description |
|--------|----------|-------------|-------------|
| Fig. 1 | `NDSI_Distribution_Validation.png` | `05_ndsi_threshold_validation.py` | NDSI distribution for snow vs non-snow pixels; quantile lines and approximate ROC threshold |
| Fig. 2 | `Snow_Accuracy_Yearly.png` | `05_ndsi_threshold_validation.py` | Per-year snow classification accuracy (OA, UA, PA) with trend lines |
| Fig. 3 | `SCF_Threshold_Analysis.png` | `06_scf_zone_classification.py` | 4-panel: SCF distribution, low/high range detail, zone pie chart |
| Fig. 4 | `SCF_Combined_4Panel.png` | `09_scf_overview_figure.py` | 4-panel: mean SCF map, trend map with marginal profiles, zoom panels |
| Fig. 5 | `PSA_Trend.png` | `07_snow_area_trend_figure.py` | PSA trend map + violin plots by elevation and latitude |
| Fig. 6 | `SSA_Trend.png` | `07_snow_area_trend_figure.py` | SSA trend map + violin plots (set `ANALYSIS_TYPE = "SSA"`) |
| Fig. 7 | `SER_Trend_Analysis.png` | `08_ser_trend_figure.py` | 5-panel: SER trend map, violin plots, SLA scatter, SLA box plot |

## Reproducing Figures

```bash
# Install dependencies
pip install -r requirements.txt

# Authenticate GEE (for scripts 07-09)
earthengine authenticate

# Example: regenerate SCF overview figure
python code/09_scf_overview_figure.py
```

All figures are saved at 300 dpi with white background, using Times New Roman font throughout for consistency with the manuscript.
